## 🗺️ What is SecArch Lab?

SecArch Lab is a **public, open-source library** of interactive security architecture diagrams built diagram by diagram and published via GitHub Pages. Every diagram is a fully interactive widget — components are clickable, flows are animated, and every element links to a detailed explanation.

> **No frameworks. No build tools. No dependencies.**
> Pure HTML + CSS + JavaScript. One file per diagram. Drop into a repo, live in 60 seconds.

## ⚙️ How Each Diagram Works

Every diagram page is a **self-contained interactive HTML widget** with:

```
┌─────────────────────────────────────────────────────┐
│  ▶ Trace flow button  ──→  Animates request path    │
│  🖱 Click any component ─→  Opens detail panel      │
│  📋 Detail panel     ──→  Description + bullet list │
│  🔍 Reset button      ──→  Clears all highlights    │
└─────────────────────────────────────────────────────┘
```

**Tech used:**
- Pure HTML + CSS + JavaScript — no React, no Vue, no framework
- Google Fonts (Syne · DM Sans · JetBrains Mono)
- Zero npm packages, zero build step
- GitHub Pages for hosting — auto-deploys on push

<br>

## 🚀 Adding a New Diagram

When a new diagram is ready:

**1.** Upload the new `.html` file to this repo

**2.** Open `index.html` and find the diagram's entry — change:
```js
// Before
{ id: 'netsec', live: false, ... }

// After
{ id: 'netsec', live: true, link: 'network-security.html', ... }
```

**3.** Commit — GitHub Pages deploys in ~60 seconds

The card on the library page automatically switches from grey `Planned` to green `Live now`.
